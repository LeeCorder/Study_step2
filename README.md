## 1. Semantic Tag (시맨틱태그)
* &lt;header&gt;
* &lt;nav&gt;
  * 사이트의 주요 목차 및 메뉴 등을 포함
  * 일반적으로 footer 영역에는 사용하지 않음
* &lt;main&gt;
  * 문서의 핵심
  * header, nav, article, asdie, footer 요소의 자식으로 불가능
* &lt;section&gt;
* &lt;article&gt;
* &lt;aside&gt;
* &lt;footer&gt;
